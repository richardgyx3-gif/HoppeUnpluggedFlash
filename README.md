# Hoppe语录闪卡学习网站

基于Hans-Hermann Hoppe教授政治哲学语录的交互式学习网站，提供闪卡式的深度思考体验。

## 📖 概述

本网站提取了Hoppe教授的经典语录，按照问题-答案的形式组织成56张学习闪卡，涵盖8个主要政治哲学主题。网站采用学术暗黑风格设计，为深度阅读和思考提供最佳环境。

## 🎯 功能特性

### 核心功能
- **📚 闪卡学习**: 56张精心整理的哲学语录卡片
- **🎲 随机抽取**: 随机显示语录，激发思考
- **🔍 智能搜索**: 支持问题和内容的全文搜索
- **❤️ 收藏系统**: 标记重要的语录随时回顾
- **📱 响应式设计**: 完美适配桌面端和移动端

### 导航功能
- **📑 章节导航**: 按8个主题分类浏览
- **⚡ 快速切换**: 上下翻页、随机跳转
- **⌨️ 键盘快捷键**: 提高学习效率

### 界面特色
- **🌙 学术暗黑模式**: 减少视觉疲劳，提升专注度
- **📖 书籍式布局**: 类似实体书的阅读体验
- **✨ 优雅动效**: 流畅的交互反馈

## 🗂️ 内容结构

### 8个主要章节

1. **国家的本质** (7张卡片)
   - 探讨国家的性质和存在的合理性

2. **政府与历史** (5张卡片)
   - 比较不同政治制度的历史表现

3. **国家活动** (9张卡片)
   - 分析政府各项职能的作用

4. **当前问题** (9张卡片)
   - 针对当代政治问题的思考

5. **私法社会** (10张卡片)
   - 提出无国家社会的可能性

6. **未来** (3张卡片)
   - 对政治发展方向的前瞻

7. **智力发展** (3张卡片)
   - Hoppe个人的思想发展历程

8. **推荐阅读** (10张卡片)
   - 相关书籍和资料推荐

## 🚀 使用指南

### 基本操作
- **翻页**: 使用"上一张"/"下一张"按钮或键盘左右箭头键
- **随机抽取**: 点击"随机抽取"按钮或按空格键/Enter键
- **搜索**: 在搜索框输入关键词查找相关内容
- **收藏**: 点击星形图标收藏当前语录

### 键盘快捷键
| 按键 | 功能 |
|------|------|
| `←` `→` | 上一张/下一张 |
| `空格` `Enter` | 随机抽取 |
| `F` | 切换收藏状态 |
| `S` | 聚焦搜索框 |
| `Esc` | 清除搜索/关闭菜单 |

### 移动端操作
- 点击左上角菜单图标打开章节导航
- 支持触摸手势操作
- 响应式布局适配小屏幕
